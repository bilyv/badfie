## ğŸš€ Quick Start

### Prerequisites

- **Node.js** (v18 or higher) - [Download here](https://nodejs.org/)
- **Bun** (recommended) - [Install Bun](https://bun.sh/)

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repository-url>
   cd digitalstock
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Start development server**
   ```bash
   bun run dev
   ```

4. **Open your browser**
   ```
   http://localhost:8080
   ```
